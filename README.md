## Обо мне

Программирую со старших классов. Математическую подготовку и практику работы с данными применяю в системном анализе: структурирую информацию, проверяю связи между требованиями, данными и поведением системы.

Практиковал PostgreSQL и SQLite на учебной базе авиаперевозок: JOIN, фильтрацию, индексы, функции, транзакции, уровни изоляции и блокировки. Использую Python для вычислений, анализа данных и прототипирования; публиковал учебные пакеты на [PyPI](https://pypi.org/user/Boo4kin/).

## Дипломный проект

### [Attention Meta-Learner for MOEX Si Futures](https://github.com/Boo4kin/attention-meta-learner-moex)

Выпускная квалификационная работа 2026 года: адаптивный ансамбль для краткосрочного прогнозирования направления цены фьючерса Si на Московской бирже.

- **Данные:** 219 529 минутных OHLCV-баров, 29 признаков
- **Модели:** CatBoost, LightGBM, KAN-proxy, Mamba-proxy и attention-мета-модель
- **Проверка:** временной holdout на 7 550 наблюдениях, purged cross-validation и векторизованный бэктест
- **Результат:** воспроизводимый исследовательский пайплайн, ноутбук, тесты и Streamlit-дашборд

[Диплом PDF](https://github.com/Boo4kin/attention-meta-learner-moex/blob/main/docs/thesis.pdf) · [Презентация](https://github.com/Boo4kin/attention-meta-learner-moex/blob/main/docs/presentation.pdf) · [Исследовательский ноутбук](https://github.com/Boo4kin/attention-meta-learner-moex/blob/main/notebooks/attention_meta_learner.ipynb)

> Метрики относятся к исследовательскому бэктесту с допущениями проекта и не являются подтвержденной доходностью реальной торговой системы.
